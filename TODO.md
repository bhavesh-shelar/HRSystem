# HR System

A full-stack HR Management System built with React, Node.js, Express, and MySQL.

## Features

- Employee Management
- Department Management
- Attendance Tracking
- Leave Management
- Admin Management

## Tech Stack

- **Frontend:** React, CSS
- **Backend:** Node.js, Express
- **Database:** MySQL (via XAMPP)

## Getting Started

1. Start XAMPP and ensure MySQL is running
2. Run `npm start` in the server directory
3. Run `npm start` in the client directory
4. Login with default credentials: admin / admin123
